# Code of Conduct

## Our Pledge

We want this project to be a welcoming place for people building agent memory,
retrieval, and developer tooling. We value curiosity, patience, and clear
technical disagreement without personal attacks.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, while still asking for evidence.
- Keep feedback focused on the work and its impact.
- Help make discussions understandable to newcomers.
- Avoid sharing private data, credentials, or personal information.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Personal attacks or sustained hostile comments.
- Publishing private information without permission.
- Deliberately derailing technical discussions.
- Encouraging unsafe handling of credentials or private runtime memory.

## Enforcement

Maintainers may remove comments, close issues, block accounts, or take other
reasonable action to protect the project community. Reports can be raised in a
GitHub issue if appropriate, or privately through the repository's available
security/contact channels when the report includes sensitive information.
