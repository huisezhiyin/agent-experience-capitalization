# Support

Agent Experience Capitalization is an early-stage open source prototype.

## Where To Ask

- Use GitHub issues for bugs, reproducible failures, and feature requests.
- Use discussions, if enabled, for design questions and open-ended ideas.
- For security issues, follow `SECURITY.md`.

## What To Include

- The command you ran.
- The expected behavior.
- The actual behavior.
- A minimal synthetic example if runtime memory or traces are involved.
- Whether Milvus Lite is installed and enabled.

Please avoid sharing private `.agent-memory/` contents, personal paths, API
keys, or private project logs.
